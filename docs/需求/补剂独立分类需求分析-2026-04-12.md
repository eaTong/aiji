# 知识库「补剂」独立分类需求分析

**日期**：2026-04-12
**版本**：v1.0
**状态**：待确认
**类型**：需求拓展分析

---

## 一、背景回顾

### 现有知识库结构（来源：docs/superpowers/specs/2026-04-12-aiji-miniprogram-design.md）

当前「身体营养」分类下的「补剂指南」子分类内容：
- 蛋白粉 / 肌酸 / BCAA / 谷氨酰胺 / 咖啡因
- 使用时机与剂量

当前「AI辟谣」分类下的「补剂误区」内容：
- "肌酸有副作用" / "补剂都是激素" / "运动完立刻喝蛋白粉才有效"

### 用户画像
- **入门健身者（A）**：刚开始健身，需要大量引导、计划和教育
- **中级健身爱好者（B）**：有一定经验，需要优化训练、精细追踪数据

---

## 二、分析结论总览

### Q1: 补充分类应该包含哪些子分类？

**建议子分类结构**（按使用目的分类）：

```
补剂
├── 增肌支持类
├── 减脂辅助类
├── 促进恢复类
├── 基础健康优化类
└── 训练周期专用类
```

### Q2: 每个子分类应该覆盖哪些具体内容？

详见本文档第三章。

### Q3: 内容难度如何分级？

采用双标签制：`入门` / `进阶`

- **入门级**：零基础用户可理解，无需背景知识
- **进阶级**：需要一定健身经验，涉及较细的概念

### Q4: 与现有「身体营养 → 补剂指南」的关系如何处理？

**建议：独立为一级分类**

理由：
1. 补剂内容体量足够大，单独成体系便于用户浏览学习
2. 补剂是用户高频提问领域，独立分类提升可发现性
3. 未来扩展空间大（更多补剂品种、循证研究等）
4. 与「AI辟谣 → 补剂误区」整合更顺畅

**过渡方案**：原「身体营养 → 补剂指南」内容迁移至新分类 → 保留入口跳转或显示"已迁移"提示

### Q5: AI辟谣中「补剂误区」内容如何整合？

**建议：整合到新分类内，作为各补剂子类的「常见误区」板块**

-辟谣内容与对应补剂知识绑定，而非单独区块
- 用户在了解某补剂时，自然看到相关辟谣内容
- 原「AI辟谣 → 补剂误区」条目调整为交叉引用形式

### 3.1 增肌支持类

**定位**：帮助用户最大化肌肉合成的补剂

#### 蛋白粉

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 什么是蛋白粉，为何比食物吸收快 | 乳清/酪蛋白/植物蛋白的吸收速率差异及适用场景 |
| 主要类型 | 乳清蛋白的分类（浓缩/分离/水解） | 水解乳清的肽分子量分析，各类型性价比计算 |
| 使用方法 | 训练后30分钟内冲泡 | 蛋白质合成窗口期科学依据，不同体重的剂量计算 |
| 常见误区 | "蛋白粉伤肾"辟谣 | 肾功能正常人群的蛋白质上限研究 |
| 注意事项 | 乳糖不耐受选择建议 | 与其他补剂的时间间隔搭配 |

#### 肌酸

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 肌酸提升力量和爆发力的原理 | ATP-PCr能量系统的深层机制 |
| 使用方法 | 简单的每天5g维持期方案 | 五天饱和法 vs 日常维持期的效果对比研究 |
| 常见误区 | "肌酸有副作用/导致肾结石"辟谣 | 肌酸与水分滞留的真相，是否需要冲击期 |
| 注意事项 | 适用人群（青少年/孕妇需咨询医生） | 不同形式的肌酸（一水肌酸 vs HCl肌酸）效果差异 |
| 产品选择 | 如何识别正规品牌 | 纯度检测与性价比分析 |

#### BCAA

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | BCAA是什么，为何被称为"肌肉保护剂" | 亮氨酸/异亮氨酸/缬氨酸的独立作用机制 |
| 使用方法 | 训练前还是训练后喝 | 与蛋白粉相比的真实功效差异研究 |
| 常见误区 | "BCAA比蛋白粉更值得买"辟谣 | 蛋白质摄入充足时BCAA的边际效益分析 |
| 注意事项 | 适用场景（空腹训练/低蛋白饮食者） | 特定人群（肝病患者）的使用限制 |

### 3.2 减脂辅助类

**定位**：帮助优化脂肪代谢的补剂

#### 咖啡因

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 咖啡因如何提升代谢和专注力 | 腺苷受体拮抗与脂肪氧化的关系 |
| 使用方法 | 训练前30分钟的最佳摄入量 | 不同体重/咖啡因敏感度的剂量个性化方案 |
| 常见误区 | "咖啡因完全无害"和"咖啡因完全有害"辟谣 | 长期大量摄入的耐受性研究 |
| 注意事项 | 每日安全剂量（400mg FDA标准） | 与其他补剂（如绿咖啡提取物）的叠加效果 |
| 产品选择 | 氮泵vs纯咖啡因粉的利弊 | 天然vs合成咖啡因的差异 |

#### 左旋肉碱

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 左旋肉碱在脂肪代谢中的作用 | 长链脂肪酸转运入线粒体的机制 |
| 使用方法 | 服用时机（有氧前30-60分钟） | 为何需要配合运动才有效 |
| 常见误区 | "服用肉碱就能躺着减肥"辟谣 | 内源性肉碱合成与外源补充的关系 |
| 注意事项 | 素食者/严格减脂人群的适用性 | 产品形式（液体制剂vs胶囊吸收率） |

### 3.3 促进恢复类

**定位**：帮助身体从训练中恢复的补剂

#### 谷氨酰胺

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 谷氨酰胺对免疫系统的支持作用 | 肠道屏障完整性维护与谷氨酰胺的关系 |
| 使用方法 | 大强度训练后的补充时机 | 何时真正需要（一般饮食可满足） |
| 常见误区 | "谷氨酰胺是增肌必需品"辟谣 | 普通人是否需要额外补充的科学证据 |
| 注意事项 | 特殊人群（肝/肾功能问题）的使用 | 与其他氨基酸的相互作用 |

#### Omega-3 鱼油

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | EPA/DHA对炎症反应调节的作用 | Omega-3/Omega-6比例对慢性炎症的影响 |
| 使用方法 | 每日建议摄入量（EPA+DHA 1-3g） | 与食物同服的吸收率优化 |
| 常见误区 | "鱼油可以完全替代鱼类摄入"辟谣 | 纯度与浓度的差异，低质量鱼油的风险 |
| 注意事项 | 服用抗凝药物者的注意事项 | 鱼油与维生素D的协同作用 |
| 产品选择 | rTG vs EE vs TG 形式的吸收率 | 第三方检测认证（IFOS等）的重要性 |

### 3.4 基础健康优化类

**定位**：优化整体健康状态的微量营养补剂

#### 维生素D

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 维生素D对骨骼和肌肉功能的影响 | 维生素D受体与肌肉蛋白合成的关系 |
| 使用方法 | 每日建议摄入量（1000-4000IU） | 如何通过血液检测优化剂量 |
| 常见误区 | "晒太阳就足够"辟谣 | 地理位置/肤色/防晒霜对维生素D合成的影响 |
| 注意事项 | 与维生素K2的协同作用 | 维生素D中毒的防范 |

#### 镁

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 镁在肌肉收缩和放松中的作用 | 镁与ATP合成、激素调节的关系 |
| 使用方法 | 睡前服用的睡眠优化效果 | 不同形式（氧化镁/柠檬酸镁/甘氨酸镁）的吸收率 |
| 常见误区 | "只要吃蔬菜就不会缺镁"辟谣 | 现代饮食中镁含量下降的原因 |
| 注意事项 | 运动量大者的镁流失 | 与钙的配比建议 |

#### 锌

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 锌对睾酮合成和免疫功能的作用 | 锌参与300+酶反应的机制 |
| 使用方法 | 每日建议摄入量（11mg男性） | 补锌的最佳时机（避免与钙铁同服） |
| 常见误区 | "锌可以随意补充"辟谣 | 长期过量补充的副作用（铜缺乏） |
| 注意事项 | 素食者/高训练量人群的易缺乏性 | 与维生素D的协同 |

### 3.5 训练周期专用类

**定位**：针对特定训练阶段或时机使用的补剂

#### 氮泵（训练前）

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 氮泵的"泵感"来源（一氧化氮扩张血管） | 精氨酸/瓜氨酸的作用机制研究 |
| 使用方法 | 训练前20-30分钟 | 不同成分组合（咖啡因+精氨酸+β丙氨酸）的效果分析 |
| 常见误区 | "氮泵无害可长期大量使用"辟谣 | 咖啡因依赖性和耐受性研究 |
| 注意事项 | 心血管疾病患者的禁忌 | 晚上训练避免服用的原因 |
| 产品选择 | 成分表解读，识别无效成分 | 性价比与有效剂量的关系 |

#### 训练后恢复冲剂

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | 训练后营养窗口期的概念 | 蛋白质/碳水配比与肌肉糖原恢复的关系 |
| 使用方法 | 训练后尽快补充 | 与完整正餐的时间安排 |
| 常见误区 | "必须立刻喝否则肌肉会分解"辟谣 | 窗口期的真实时长（1-2小时vs更宽裕） |
| 注意事项 | 碳水来源的选择（麦芽糊精vs真实食物） | 热量计算，避免热量过剩 |

#### β-丙氨酸

| 内容维度 | 入门 | 进阶 |
|---------|------|------|
| 定义与原理 | β-丙氨酸与组胺酸合成卡尼汀的作用 | 提高肌肉卡尼汀储备延缓疲劳的机制 |
| 使用方法 | 每日2-5g分次服用（减少刺痛感） | 长期使用的效果评估周期 |
| 常见误区 | "吃了几天没感觉就是没用"辟谣 | 为何需要2-4周才能感受到效果 |
| 注意事项 | 刺痛感（paresthesia）是无害的 | 与肌酸的叠加效果 |

## 四、内容深度建议

### 双标签体系

| 标签 | 定义 | 触发条件 |
|------|------|---------|
| `入门` | 零基础用户即可理解，无需任何健身/营养背景知识 | 新用户引导完成后默认展示 |
| `进阶` | 需要一定健身经验，涉及较细节的概念和计算 | 中级用户或明确点击"查看更多"时展示 |

### 内容深度梯度设计

```
入门内容 → 建立正确认知
├── 是什么（补剂的定义）
├── 为什么（补剂的作用原理，用比喻解释）
├── 什么时候用（基本使用时机）
└── 常见误区（最常听到的错误观念）

进阶内容 → 精细化应用
├── 深度原理（生化机制）
├── 个性化剂量（根据体重/训练量计算）
├── 产品选择（成分表解读/品牌比较）
├── 组合策略（与其他补剂的协同）
└── 循证评估（研究数据解读）
```

### AI推荐触发场景

| 用户行为 | 推荐内容 |
|---------|---------|
| 用户询问"该不该吃蛋白粉" | 增肌支持类 → 蛋白粉（入门） |
| 用户完成首次力量训练记录 | 增肌支持类 → 肌酸（入门） |
| 用户表示想减脂 | 减脂辅助类 → 咖啡因/左旋肉碱（入门） |
| 用户询问补剂是否有副作用 | 对应补剂子类的「常见误区」辟谣内容 |
| 中级用户查看补剂分类 | 展示全部「入门+进阶」内容 |

## 五、与现有内容的关系处理建议

### 5.1 现有「身体营养 → 补剂指南」迁移方案

| 原内容条目 | 迁移至新分类位置 |
|-----------|-----------------|
| 蛋白粉 | 补剂 → 增肌支持类 → 蛋白粉 |
| 肌酸 | 补剂 → 增肌支持类 → 肌酸 |
| BCAA | 补剂 → 增肌支持类 → BCAA |
| 谷氨酰胺 | 补剂 → 促进恢复类 → 谷氨酰胺 |
| 咖啡因 | 补剂 → 减脂辅助类 → 咖啡因 |

**迁移后**：「身体营养 → 补剂指南」子分类移除或保留为空分类并提示"内容已迁移至补剂分类"

### 5.2 现有「AI辟谣 → 补剂误区」整合方案

| 原辟谣条目 | 整合至新分类位置 |
|-----------|-----------------|
| "肌酸有副作用" | 补剂 → 增肌支持类 → 肌酸 → 常见误区 |
| "补剂都是激素" | 补剂 → 概论（什么是补剂/补剂的安全性） |
| "运动完立刻喝蛋白粉才有效" | 补剂 → 增肌支持类 → 蛋白粉 → 常见误区 |

**整合后**：「AI辟谣 → 补剂误区」分类可选择：
- 方案A：完全移除，辟谣内容已融入各子类
- 方案B：保留分类但仅保留非补剂类辟谣内容，补剂相关做交叉引用

### 5.3 新旧分类结构对比

**修订前**：
```
知识库
├── 身体营养
│   └── 补剂指南（蛋白粉/肌酸/BCAA/谷氨酰胺/咖啡因）
├── ...
└── AI辟谣
    └── 补剂误区
```

**修订后**：
```
知识库
├── 身体营养（补剂相关内容移除）
├── 补剂 ⭐（新增）
│   ├── 增肌支持类（蛋白粉/肌酸/BCAA）
│   ├── 减脂辅助类（咖啡因/左旋肉碱）
│   ├── 促进恢复类（谷氨酰胺/Omega-3）
│   ├── 基础健康优化类（维生素D/镁/锌）
│   └── 训练周期专用类（氮泵/恢复冲剂/β-丙氨酸）
├── ...
└── AI辟谣（补剂误区相关内容整合或移除）
```
