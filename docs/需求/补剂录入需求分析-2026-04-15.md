# 补剂录入需求分析

> **日期：** 2026-04-15
> **功能：** 补剂情况录入 + CHITCHAT 补剂上下文
> **优先级：** P1

---

## 一、背景与目标

### 1.1 问题

健身人群常使用补剂（蛋白粉、肌酸、BCAA等），但当前系统没有记录补剂使用情况。AI 教练在闲聊时无法参考用户的补剂习惯给出更专业的建议。

### 1.2 目标

- 用户可以记录每日补剂摄入情况
- AI 教练在闲聊时可以参考用户的补剂信息
- 补剂记录作为用户健康档案的一部分

---

## 二、数据模型

### 2.1 SupplementRecord 模型

**文件：** `backend/prisma/schema.prisma`

```prisma
model SupplementRecord {
  id          String   @id @default(uuid())
  userId      String
  date        DateTime @db.Date
  supplement  String   // 补剂名称
  dosage      String?  // 用量描述，如 "30g"、"2粒"
  timing      String?  // 服用时间：早餐后、训练前、睡前等
  note        String?  // 备注
  createdAt   DateTime @default(now())

  user        User     @relation(fields: [userId], references: [id])

  @@unique([userId, date, supplement])
}
```

### 2.2 常用补剂列表

| 补剂 | 说明 |
|------|------|

| 谷氨酰胺 | Glutamine |
| 鱼油 | Fish Oil |

## 三、API 设计

### 3.1 记录补剂

**端点：** `POST /api/supplements`

**请求体：**
```typescript
interface CreateSupplementRecord {
  supplement: string   // 补剂名称
  dosage?: string      // 用量
  timing?: string      // 服用时间
  note?: string        // 备注
  date?: string        // 日期，默认今天
}
```

**响应：**
```typescript
interface SupplementRecordResponse {
  id: string
  supplement: string
  dosage: string | null
  timing: string | null
  date: string
  note: string | null
}
```

### 3.2 获取补剂记录

**端点：** `GET /api/supplements`

**查询参数：**
| 参数 | 类型 | 说明 |
|------|------|------|
| date | string | 日期 (YYYY-MM-DD)，默认今天 |
| days | number | 获取最近 N 天的记录，默认 7 |

### 3.3 删除补剂记录

**端点：** `DELETE /api/supplements/:id`

## 五、AI 上下文集成

### 5.1 buildUserContext 扩展

在 `buildUserContext` 函数中添加补剂信息：

```typescript
// 获取最近补剂记录
const recentSupplements = await prisma.supplementRecord.findMany({
  where: { userId },
  orderBy: { date: 'desc' },
  take: 5,
})

let supplementText = '暂无补剂记录'
if (recentSupplements.length > 0) {
  supplementText = recentSupplements.map(r =>
    `${r.date.toISOString().split('T')[0]} - ${r.supplement}${r.dosage ? ` (${r.dosage})` : ''}`
  ).join('\n')
}
```

### 5.2 用户上下文输出格式

```
补剂使用情况：
蛋白粉 30g 早餐后
肌酸 5g 训练前
(最近5条记录)
```
