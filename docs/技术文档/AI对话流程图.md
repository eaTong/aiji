```

                                            ▼
```

```
┌─────────────────┬──────────────────────────────────────────────────┐

├─────────────────┼──────────────────────────────────────────────────┤
│ USE_DEFAULTS    │ 使用默认值继续（unit=kg, date=今天）              │
│ OFFER_OPTIONS   │ 提供选项让用户选择                                │
│ CANCEL_INTENT   │ 取消意图，友好提示"好的，有需要随时叫我"          │
└─────────────────┴──────────────────────────────────────────────────┘
```

## 十、数据模型

```
┌─────────────────────────────────────────────────────────────────────┐
│                          核心数据模型                                 │
└─────────────────────────────────────────────────────────────────────┘

ChatSession (会话)
├── id: String (UUID)
├── userId: String
├── status: Enum (ACTIVE, AWAITING_CLARIFICATION, COMPLETED, EXPIRED)
├── context: Json { originalMessage, extractedEntities }
├── pendingIntent: String?
├── missingFields: Json?
├── clarificationHistory: Json?
├── retryCount: Int
├── createdAt: DateTime
└── updatedAt: DateTime

ChatMessage (消息)
├── id: String (UUID)
├── userId: String
├── role: Enum (user, ai)
├── type: Enum (text, card)
├── content: String?
├── cardType: String?
├── cardData: Json?
├── actions: Json?
├── planId: String?
├── sessionId: String?
├── createdAt: DateTime
└── updatedAt: DateTime

PushQueue (推送队列)
├── id: String (UUID)
├── userId: String
├── type: String (morning-report, weekly-report, overtraining-warning)
├── title: String
├── content: String
├── data: Json?
├── isRead: Boolean
├── createdAt: DateTime
└── sentAt: DateTime?
```
