## 二、导入流程

```
┌──────────────────────────────────────────────────────────────┐
│                     完整导入流程                              │
├──────────────────────────────────────────────────────────────┤
│  1. download    │ 下载 free-exercise-db 原始数据              │
│  2. transform   │ 字段转换（中英映射、分类映射、肌肉映射）      │
│  3. enrich      │ AI增强（中文说明、注意事项、易错点）         │
│  4. upload      │ 图片上传至阿里云OSS                          │
│  5. import      │ 最终数据用于项目导入                         │
└──────────────────────────────────────────────────────────────┘
```

## 四、字段映射

### 4.1 基础映射

| free-exercise-db | AI己字段 | 映射逻辑 |
|------------------|----------|----------|
| name | name | 待AI填充中文名 |
| name | nameEn | 保留英文名 |
| category | category | 肌肉推断 |
| equipment | equipment | 器械映射 |
| primaryMuscles | primaryMuscles | 中文化 |
| secondaryMuscles | secondaryMuscles | 中文化 |
| instructions | instructions | 数组join为字符串 |
| - | commonMistakes | AI搜索总结 |
| images | videoUrl | OSS图片URL |

### 4.2 分类映射

| 主肌群 | AI己分类 |
|--------|----------|
| 胸大肌 | CHEST |
| 背阔肌/斜方肌 | BACK |
| 股四头肌/腘绳肌/臀肌 | LEGS |
| 三角肌 | SHOULDERS |
| 肱二头肌/肱三头肌 | ARMS |
| 腹肌 | CORE |
| 心肺训练 | CARDIO |

### 4.3 器械映射

| free-exercise-db | AI己 |
|------------------|------|
| barbell/dumbbell/machine等 | GYM |
| body only | BODYWEIGHT |

### 4.4 肌肉中文化

17种肌肉的英中映射，详见 `src/mapping.ts`

## 五、AI增强设计

### 5.1 搜索策略

- 搜索词：`{动作英文名} exercise form tips mistakes warnings`
- 数据源：DuckDuckGo 搜索结果
- 提取内容：注意事项、易错点

### 5.2 正则匹配

**注意事项**：
```
/warning[s]?:?\s*([^.]+)/gi
/tip[s]?:?\s*([^.]+)/gi
/注意[s]?:?\s*([^。]+)/g
```

**易错点**：
```
/common\s+mistake[s]?:?\s*([^.]+)/gi
/avoid\s+([^.]+)/gi
/不要[s]?:?\s*([^。]+)/g
```

### 5.3 并发控制

- 同时处理5个动作
- 每个动作搜索后等待100ms避免限流

## 六、执行步骤

### 6.1 准备阶段

```bash
# 1. 进入脚本目录
cd backend/scripts/import-free-exercise-db

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 填入 OSS 配置
```

### 6.2 下载数据

```bash
# 方式1：自动下载（仅数据）
npm run download

# 方式2：手动下载图片
cd data
git clone --depth 1 --filter=blob:none --sparse \
  https://github.com/yuhonas/free-exercise-db.git temp_repo
cd temp_repo
git sparse-checkout set exercises
cp -r exercises/* ../images/
cd ..
rm -rf temp_repo
```

### 6.3 转换数据

```bash
npm run transform
```

### 6.4 AI增强（耗时最长）

```bash
npm run enrich
# 预计30-60分钟处理800+动作
```

### 6.5 上传图片

```bash
npm run upload
```

### 6.6 一键执行

```bash
npm run import
```
