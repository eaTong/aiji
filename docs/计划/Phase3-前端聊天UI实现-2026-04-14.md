**新建文件**：
```
components/chat/
├── ChatMessageList.vue      # 消息列表容器（scroll-view）
├── ChatMessage.vue           # 单条消息（文本/卡片）
├── ChatInput.vue             # 输入框组件
├── QuickButtons.vue          # 快捷按钮
```

**改动**：
- `pages/index/index.vue` - 引用拆分后的组件

### Step 3: 快捷按钮实现

**目标**：实现首页底部快捷按钮

**设计**（根据规格只有 2 个按钮）：
| 按钮 | 触发 | 行为 |
|------|------|------|
| 练什么 | 发送"今天练什么好" | 调用 `/api/chat/send` |
| 记录 | 发送"记录训练" | 调用 `/api/chat/send` |

**快捷输入预设**：
```
- "记体重 {重量}kg"
- "今天练了 {动作} {重量}kg {次数}个"
- "状态怎么样"
```

---

### Step 4: 主动推送处理

**目标**：处理 `/api/chat/init` 返回的未读推送

**流程**：
1. 用户打开首页 → 调用 `/api/chat/init`
2. 获取 `greeting` 和 `unreadPushes`
3. 将未读推送插入消息流顶部
4. 用户点击推送 → 调用 `/api/chat/action`

**新建**：
```
stores/push.ts
- unreadPushes: Push[]
- fetchPushes()
- markAsRead(pushId)
```

---

### Step 5: 卡片动作处理

**目标**：完善 `stores/chat.ts` 中的动作处理

**action 类型**：
| action | 处理 |
|--------|------|
| `navigate` | `uni.navigateTo({ url: target })` |
| `dismiss` | 移除卡片 |
| `confirm` | 调用 `/api/chat/confirm` 保存 |
| `cancel` | 调用 `/api/chat/confirm` 取消 |
| `regenerate` | 重新发送消息 |
| `share` | 调用 `uni.share()` |
